# MNIST Handwritten Digit Classification

## 📌 Overview

This project implements a deep learning model to classify handwritten digits (0–9) using the MNIST dataset.

The model is built using **TensorFlow and Keras** and achieves high accuracy in recognizing digit patterns from grayscale images.

---

## Features

* Classifies handwritten digits (0–9)
* Uses neural networks for prediction
* Trained on the MNIST dataset
* High accuracy on test data
* Simple and efficient implementation
