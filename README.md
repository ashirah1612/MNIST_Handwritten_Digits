# MNIST Handwritten Digit Classification

Neural network for handwritten digit classification (MNIST) using Python, TensorFlow & Keras.

## How to Run
- Open `handwritten_MNIST.ipynb` in VS Code or Colab
- Run all cells
- Required Python packages: tensorflow, matplotlib, numpy
